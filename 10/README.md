# Подготовка кода
Запустить проект (создать виртуальное окружение с установкой всех необходимых библиотек, компиляция кода и установка модуля, запуск линтеров):
```
make
```
# Запустить unittests и тест производительности
```
make test
```
## Удалить venv
```
make clean
```
